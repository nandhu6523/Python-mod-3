## SEB - SWAP TUPLES

### AIM  

To write a Python program to swap given tuples.

### ALGORITHM

1. Begin the program.  
2. Input tuple1 from the user using eval().
3. Input tuple2 from the user using eval().
4. Print tuple2.
5. Print tuple1.  
6. Terminate the program.

### PROGRAM

```
tuple1 = eval(input())
tuple2 = eval(input())
print(tuple2)
print(tuple1)

```
